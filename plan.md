# Sahakar Celebrations Portal - Master Project Index

Welcome to the master roadmap and architectural specifications index for the **Sahakar Celebrations Portal**. This repository is configured to build a world-class cinematic, emotional, and luxury wedding invitation experience.

---

## 📂 Master Documentation Suite
Detailed specifications and implementation details are divided into the following documents inside the `/docs` directory:

1. **[Product Requirements Document (PRD)](file:///e:/K4NN4N/sahakar-celebrations/docs/PRD.md)**
   * Describes the branding vision, target user segments, MVP features (Splash screen, particle dynamics, continuous audio), and product constraints.
2. **[Technical Requirements Document (TRD)](file:///e:/K4NN4N/sahakar-celebrations/docs/TRD.md)**
   * Architect's guide detailing Vite, React TS, GSAP scroll triggers, Lenis scroll integration, the audio autoplay bypass algorithm, and performance optimization targets.
3. **[UI/UX Design Document](file:///e:/K4NN4N/sahakar-celebrations/docs/UI_UX_Design.md)**
   * Articulates design tokens (Obsidian Dark `#0D0C10`, Champagne Gold `#DFBA73`), custom Cinzel typography rules, responsive cards, canvas-based particles, and interactive cursor glow elements.
4. **[Appflow & Navigation Document](file:///e:/K4NN4N/sahakar-celebrations/docs/Appflow.md)**
   * Charts the guest scroll journey, visual sequence transitions, audio toggles state machines, and responsive rules. Includes a detailed Mermaid diagram.
5. **[Backend Schema Document](file:///e:/K4NN4N/sahakar-celebrations/docs/Backend_Schema.md)**
   * Formulates Neon serverless PostgreSQL SQL definitions (DDL queries), indices, unique key constraints, Drizzle ORM model schemas, and API REST specs.
6. **[Implementation Plan](file:///e:/K4NN4N/sahakar-celebrations/docs/Implementation_Plan.md)**
   * Formulates a step-by-step 7-phase timeline covering bootstrapping local scripts, implementing GSAP effects, deploying SQL tables, routing API dispatches, and deploying to Vercel.
